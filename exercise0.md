# Exercise 0: Environment and Libraries

## Objective
Set up the environment for Python development.

## Steps Completed

### 1. Install Python
Python is already installed on the system.

### 2. Verify Installation
```bash
$ python3 --version
Python 3.12.3
```

### 3. Hello World Program
Created `hello_world.py`:

```python
#!/usr/bin/env python3

print("Hello, World!")
```

### 4. Create Virtual Environment
```bash
$ python3 -m venv venv
$ source venv/bin/activate
(venv) $ python --version
Python 3.12.3
```

### 5. Execute Hello World
```bash
(venv) $ python hello_world.py
Hello, World!
```

## Result
Environment is ready for prompt pattern exercises with Python 3.12.3 installed in virtual environment.
